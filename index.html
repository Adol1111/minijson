MiniJSON

MiniJSON for Java

MiniJSON is a minimal JSON parser/generator library in Java. Please check out the project on GitHub: Minimal JSON Builder - Parser Library in Java.

Fork me on GitHub!

New Kind of JSON Library

MiniJSON was created because I needed a "minimal" JSON builder/parser that did not carry the whole baggage of object mapping/data binding, etc. Eventually, the library grew, and it now includes many different "API sets" (mostly, for ease of use), which you cannot find in any other JSON libraries. Please give it a try and find out for yourself how much more you can do with new JSON APIs.

Goals

  1. To make JSON parsing easier and simpler.
  2. To make JSON building easier and simpler.

Features

  • It has no dependency on third party libraries other than JDK.
  • It is light-weight. It does not try to do more than necessary.

API Design

MiniJSON provides simple and yet flexible APIs. It's easy to use the library in your apps thanks to this fliexible APIs. Please refer to the API docs for more information.

Usage

If you use Maven, you can build and (locally) install the MiniJSON library using standard Maven goals. Or, you can specify the Maven project dependency as follows: 


  org.aerysoft.minijson
  minijson-core
  0.9.1

Sample Code

MiniJSON provides many "advanced" and flexible sets of APIs, including "partial" JSON parsing and building. It also provides more conventional APIs which are generally found in other JSON libraries. For example, you can to the following using the "mini" wrapper classes:

JsonParser parser = new MiniJsonParser(); String json = "{\"a\":\"b\"}"; Object obj = parser.parse(json); JsonBuilder builder = new MiniJsonBuilder(); Object obj = ...; String json = builder.build(obj, 4);
